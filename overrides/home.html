High-resolution real-time telemetry acquisition and display

McLaren ATLAS is the industry standard visualization tool in Formula One, and a vital tool for engineering projects with demanding telemetry requirements.
It is a modern desktop client for Windows, with responsive, high-resolution displays.

Analyze traces in a sophisticated waveform view, with integrated thresholds, events, alarms, band-pass filtering and functions — plus scatterplots, histograms, numeric gauges, and more than a dozen other displays.

Monitor up to 100,000 channels at rates up to 1 KHz, capturing UDP, TCP or Kafka event streams, and recording to file or shared telemetry database.

Flexible & Extensible

Extend the client with plugins to add your own displays, protocols and file formats.
Check out the developer guides, samples and API references on this site for details.

New for 2021: Integrate ATLAS into your environment using Web Services.
Supports time-series, relational, columnar, document, file and object storage —
with real-time updates and no client-side plugin.
